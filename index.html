tc-indymedia-drupal-6-build by HongPong

Drupal 6.x Indymedia install build based on Twin Cities Indymedia setup from http://twincities.indymedia.org

Based on the setup created by a team for http://twincities.indymedia.org, a Drupal distribution with the key IndyMedia and other modules included. This install saves the trouble of finding all the contrib modules, including indymedia_cities block and hidden, which is used for community moderation. Not yet released, the TCIMC current theme is being rebuilt and cleaned up to provide an easy way to present your IMC's content; however since it was created the Panels/CTools, Views and Features have all become very powerful ways to show content and should be checked out too. To copy TCIMC's Features (featured main column stories) control, use the taxonomy_role module to create a 'Features' taxonomy that is restricted to editors. (with 2 tags, Features and Non-features). Create a View showing Features-only nodes and you've got a main editorial controlled Feature area. Insert the view onto the front page with insert_view module or Panels. Some fun optional modules like feedapi and location are included as well. See /sites/all/modules for notes on what features.
